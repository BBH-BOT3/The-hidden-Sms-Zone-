# 📱 The hidden Sms Zone - গ্রুপ ইন্টিগ্রেশন সেটআপ গাইড

এই গাইডে আপনি শিখবেন কিভাবে বটকে একটি প্রাইভেট Telegram গ্রুপের সাথে সংযুক্ত করতে হয় যাতে সমস্ত OTP/কোড সেখানে পাঠানো হয়।

---

## 🎯 কেন গ্রুপ ব্যবহার করবেন?

✅ সমস্ত OTP এক জায়গায় সংরক্ষিত থাকবে
✅ আপনি সহজেই সব কোড ট্র্যাক করতে পারবেন
✅ একাধিক অ্যাডমিন থাকতে পারে
✅ সম্পূর্ণ নিরাপদ এবং প্রাইভেট
✅ কোন অতিরিক্ত খরচ নেই

---

## 📋 ধাপে ধাপে সেটআপ

### ধাপ ১: একটি প্রাইভেট গ্রুপ তৈরি করুন

1. Telegram খুলুন
2. নতুন গ্রুপ তৈরি করুন (নীল পেন্সিল আইকন → নতুন গ্রুপ)
3. গ্রুপের নাম দিন (যেমন: "The hidden Sms Zone - OTP Codes")
4. গ্রুপটি **প্রাইভেট** রাখুন (শুধুমাত্র আপনি এবং আপনার বট)

---

### ধাপ ২: আপনার গ্রুপ ID পান

**পদ্ধতি A: @userinfobot ব্যবহার করে (সহজ)**

1. Telegram এ **@userinfobot** খুঁজুন
2. `/start` পাঠান
3. আপনার গ্রুপে যান
4. @userinfobot কে গ্রুপে যোগ করুন
5. @userinfobot এ একটি মেসেজ ফরওয়ার্ড করুন
6. আপনি একটি মেসেজ পাবেন যেখানে **Group ID** থাকবে

**উদাহরণ:**
```
Group ID: -1001234567890
```

**পদ্ধতি B: @JsonDumpBot ব্যবহার করে**

1. Telegram এ **@JsonDumpBot** খুঁজুন
2. `/start` পাঠান
3. আপনার গ্রুপে যান
4. @JsonDumpBot কে গ্রুপে যোগ করুন
5. কোন মেসেজ পাঠান
6. আপনি একটি JSON পাবেন যেখানে `"chat":{"id":-1001234567890}` থাকবে

---

### ধাপ ৩: বটকে গ্রুপে যোগ করুন

1. আপনার বটকে গ্রুপে যোগ করুন
2. বটকে **অ্যাডমিন** করুন (গ্রুপ সেটিংস → অ্যাডমিন → আপনার বট)
3. নিশ্চিত করুন যে বট **মেসেজ পাঠাতে** পারে

---

### ধাপ ৪: বট কোডে গ্রুপ ID সেট করুন

`the_hidden_sms_zone_bot_v2.py` ফাইল খুলুন এবং এই লাইনটি খুঁজুন:

```python
ADMIN_GROUP_ID = -1001234567890  # আপনার গ্রুপ ID এখানে রাখুন
```

আপনার গ্রুপ ID দিয়ে প্রতিস্থাপন করুন:

```python
ADMIN_GROUP_ID = -1001234567890  # আপনার সঠিক গ্রুপ ID
```

---

### ধাপ ৫: বট টোকেন সেট করুন

একই ফাইলে এই লাইনটি খুঁজুন:

```python
TOKEN = "YOUR_BOT_TOKEN_HERE"
```

আপনার বট টোকেন দিয়ে প্রতিস্থাপন করুন:

```python
TOKEN = "123456789:ABCdefGHIjklmnoPQRstuvwxyzABCDEFGH"
```

## 🐛 সমস্যা সমাধান

### সমস্যা ১: "বট গ্রুপে মেসেজ পাঠাচ্ছে না"

**সমাধান:**
- নিশ্চিত করুন বট গ্রুপে যোগ করা আছে
- বটকে অ্যাডমিন করুন
- গ্রুপ ID সঠিক কিনা চেক করুন
- বট টোকেন সঠিক কিনা চেক করুন

### সমস্যা ২: "Invalid group ID"

**সমাধান:**
- গ্রুপ ID নেগেটিভ হওয়া উচিত (যেমন: -1001234567890)
- @userinfobot দিয়ে আবার চেক করুন
- নিশ্চিত করুন ID সঠিকভাবে কপি করা হয়েছে

### সমস্যা ৩: "Permission denied"

**সমাধান:**
- বটকে অ্যাডমিন করুন
- বটের পারমিশন চেক করুন (মেসেজ পাঠানোর অনুমতি)
- গ্রুপ সেটিংস থেকে বটকে পুনরায় যোগ করুন

## 📱 একাধিক গ্রুপে কোড পাঠানো

যদি আপনি একাধিক গ্রুপে কোড পাঠাতে চান:

```python
ADMIN_GROUP_IDS = [
    -1001234567890,  # গ্রুপ ১
    -1001234567891,  # গ্রুপ ২
    -1001234567892   # গ্রুপ ৩
]

async def send_code_to_groups(context, user_id, user_name, country, platform, number, otp):
    for group_id in ADMIN_GROUP_IDS:
        try:
            await context.bot.send_message(
                chat_id=group_id,
                text=f"📱 নতুন অনুরোধ\n👤 {user_name}\n📞 {number}\n🔐 {otp}"
            )
        except Exception as e:
            logger.error(f"গ্রুপ {group_id} এ মেসেজ পাঠাতে ত্রুটি: {e}")
```
