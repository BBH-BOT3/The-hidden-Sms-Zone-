# 🎯 The hidden Sms Zone - সম্পূর্ণ সেটআপ গাইড (বাংলায়)

এই গাইডে আমরা শুরু থেকে শেষ পর্যন্ত সবকিছু করব। প্রতিটি ধাপ অনুসরণ করুন।

---

## 📋 প্রয়োজনীয় জিনিস

আপনার কাছে থাকা উচিত:
- ✅ একটি কম্পিউটার (Windows, Mac বা Linux)
- ✅ ইন্টারনেট সংযোগ
- ✅ Telegram অ্যাপ (ফোনে বা ডেস্কটপে)
- ✅ একটি Telegram অ্যাকাউন্ট

---

# 🔧 ধাপ ১: Telegram বট তৈরি করুন

## ১.১ BotFather এ যান

1. Telegram খুলুন
2. সার্চ বারে লিখুন: `@BotFather`
3. প্রথম ফলাফলে ক্লিক করুন

![Step 1](https://imgur.com/placeholder.png)

## ১.২ নতুন বট তৈরি করুন

1. BotFather এ `/newbot` লিখুন এবং পাঠান
2. এটি বলবে "Alright! Let's create a new bot."
3. বটের নাম দিন: `The hidden Sms Zone`
4. বটের ইউজারনেম দিন: `hidden_sms_zone_bot` (অনন্য হতে হবে)

## ১.৩ বট টোকেন পান

BotFather আপনাকে একটি দীর্ঘ টোকেন দেবে:

```
123456789:ABCdefGHIjklmnoPQRstuvwxyzABCDEFGH
```

**এটি কপি করে রাখুন!** এটি খুবই গুরুত্বপূর্ণ।

---

# 📱 ধাপ ২: একটি প্রাইভেট গ্রুপ তৈরি করুন

## ২.১ নতুন গ্রুপ তৈরি করুন

1. Telegram খুলুন
2. নীল পেন্সিল আইকন ক্লিক করুন (নতুন চ্যাট)
3. "New Group" ক্লিক করুন
4. গ্রুপের নাম দিন: `The hidden Sms Zone - OTP Codes`
5. "Create" ক্লিক করুন

## ২.২ গ্রুপ প্রাইভেট করুন

1. গ্রুপ খুলুন
2. গ্রুপ নামে ক্লিক করুন (উপরে)
3. "Group Type" খুঁজুন
4. "Private" নির্বাচন করুন

## ২.৩ গ্রুপ ID পান

### পদ্ধতি A: @userinfobot ব্যবহার করে (সবচেয়ে সহজ)

1. Telegram সার্চে `@userinfobot` লিখুন
2. `/start` পাঠান
3. আপনার গ্রুপে যান
4. @userinfobot কে গ্রুপে যোগ করুন (সদস্য যোগ করুন)
5. গ্রুপে কোন মেসেজ পাঠান
6. @userinfobot আপনাকে একটি মেসেজ পাঠাবে যেখানে থাকবে:

```
Group ID: -1001234567890
```

**এই ID টি কপি করে রাখুন!**

### পদ্ধতি B: @JsonDumpBot ব্যবহার করে

1. `@JsonDumpBot` খুঁজুন
2. `/start` পাঠান
3. গ্রুপে যোগ করুন
4. কোন মেসেজ পাঠান
5. JSON ফরম্যাটে আপনি পাবেন: `"chat":{"id":-1001234567890}`

# 💻 ধাপ ৩: আপনার কম্পিউটারে সেটআপ করুন

## ৩.১ Python ইনস্টল করুন

### Windows:
1. https://www.python.org/downloads/ এ যান
2. "Download Python 3.11" ক্লিক করুন
3. ইনস্টলার চালান
4. **গুরুত্বপূর্ণ:** "Add Python to PATH" চেক করুন
5. "Install Now" ক্লিক করুন

### Mac:
```bash
brew install python3
```

### Linux:
```bash
sudo apt-get install python3 python3-pip
```

## ৩.২ ফাইল ডাউনলোড করুন

1. আমি যে সংরক্ষণাগার দিয়েছি (`the_hidden_sms_zone_v2_complete.tar.gz`) ডাউনলোড করুন
2. এটি এক্সট্র্যাক্ট করুন (ডাবল ক্লিক করুন)
3. একটি ফোল্ডার তৈরি হবে: `the_hidden_sms_zone_v2_complete`

## ৩.৩ কমান্ড প্রম্পট/টার্মিনাল খুলুন

### Windows:
1. স্টার্ট মেনু খুলুন
2. `cmd` লিখুন এবং এন্টার দিন
3. কমান্ড প্রম্পট খুলবে

### Mac/Linux:
1. অ্যাপ্লিকেশন → ইউটিলিটিজ → টার্মিনাল খুলুন

## ৩.৪ ফোল্ডারে যান

```bash
cd Downloads/the_hidden_sms_zone_v2_complete
```

(যদি ভিন্ন জায়গায় থাকে তাহলে সেই পাথ ব্যবহার করুন)

## ৩.৫ প্যাকেজ ইনস্টল করুন

```bash
pip install -r requirements.txt
```

এটি দুটি প্যাকেজ ইনস্টল করবে:
- `python-telegram-bot`
- `requests`

# 🔑 ধাপ ৪: বট কোড কনফিগার করুন

## ৪.১ বট ফাইল খুলুন

1. ফোল্ডারে যান
2. `the_hidden_sms_zone_bot_v2.py` ফাইল খুলুন (নোটপ্যাড বা যেকোনো এডিটর দিয়ে)

## ৪.২ বট টোকেন সেট করুন

এই লাইনটি খুঁজুন (প্রায় ১০৫ নম্বর লাইনে):

```python
TOKEN = "YOUR_BOT_TOKEN_HERE"
```

এটিকে এভাবে পরিবর্তন করুন:

```python
TOKEN = "123456789:ABCdefGHIjklmnoPQRstuvwxyzABCDEFGH"
```

(আপনার সঠিক বট টোকেন দিয়ে)

## ৪.৩ গ্রুপ ID সেট করুন

এই লাইনটি খুঁজুন (প্রায় ১৫ নম্বর লাইনে):

```python
ADMIN_GROUP_ID = -1001234567890
```

এটিকে এভাবে পরিবর্তন করুন:

```python
ADMIN_GROUP_ID = -1001234567890
```

(আপনার সঠিক গ্রুপ ID দিয়ে)

## ৪.৪ ফাইল সংরক্ষণ করুন

Ctrl+S (Windows/Linux) বা Cmd+S (Mac) চাপুন

# 🌐 ধাপ ৭: ২৪/৭ হোস্ট করুন (ঐচ্ছিক)

আপনার কম্পিউটার সবসময় চালু রাখতে চান না? তাহলে একটি ক্লাউড সার্ভারে হোস্ট করুন।

## অপশন A: Railway.app (সবচেয়ে সহজ)

### ধাপ ১: GitHub এ সাইন আপ করুন
1. https://github.com/signup এ যান
2. একটি অ্যাকাউন্ট তৈরি করুন

### ধাপ ২: GitHub এ রিপোজিটরি তৈরি করুন
1. https://github.com/new এ যান
2. রিপোজিটরি নাম: `the-hidden-sms-zone`
3. "Create repository" ক্লিক করুন

### ধাপ ৩: ফাইল আপলোড করুন
1. "Add file" → "Upload files" ক্লিক করুন
2. সমস্ত ফাইল নির্বাচন করুন এবং আপলোড করুন
3. "Commit changes" ক্লিক করুন

### ধাপ ৪: Railway এ ডিপ্লয় করুন
1. https://railway.app এ যান
2. GitHub দিয়ে সাইন আপ করুন
3. "New Project" → "Deploy from GitHub"
4. আপনার রিপোজিটরি নির্বাচন করুন
5. "Deploy" ক্লিক করুন

### ধাপ ৫: পরিবেশ ভেরিয়েবল সেট করুন
1. Railway ড্যাশবোর্ডে যান
2. "Variables" ট্যাব খুলুন
3. নতুন ভেরিয়েবল যোগ করুন:

```
BOT_TOKEN = আপনার_বট_টোকেন
ADMIN_GROUP_ID = আপনার_গ্রুপ_আইডি
```

4. "Deploy" ক্লিক করুন

**এখন বট ২৪/৭ চলবে!** ✅
